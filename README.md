# JavaScript Practice Problems

This is a collection of JavaScript practice problems based on a Bangla programming book. Here you'll find exercises for different chapters to practice fundamental JavaScript concepts.

## Chapters

### Chapter 2: Variables

This chapter covers JavaScript variable declaration and usage.

<details>
<summary>Click to expand Chapter 2 exercises</summary>

#### Page 35: Basic Variable Declaration
- Store birth year in a variable
- Store number of siblings in a variable
- Store class student count in a variable
- Various numeric variable exercises

#### Page 38: Variables and console.log()
- Declare age variable and print it
- Store books to read plan in variable
- Using console.log() for output

#### Page 41: String Variables
- Declare name variable
- Declare address string variable
- String data type with console.log()

#### Page 42: Boolean Variables
- Declare boolean variable for happiness status
- Boolean data type with console.log()

#### Page 47: Arithmetic Operations
- Calculate remaining money after expenses
- Basic subtraction operations with variables

</details>
